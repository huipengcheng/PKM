# [[Fonts]]

---

In arch, fonts should be installed at:
- For a single user: `~/.local/share/fonts`
- For system-wide (or using pacman): `/usr/local/share/fonts`

- Source Code Pro
- Source Han Sans
- Hack Nerd
- Apple-related
	- Helvetica Neue
	- Helvetica Now
	- New York
- DejaVu
- FiraCode
- JetBrainsMono
- OpenSans
- Emoji
	- Apple Color Emoji
	- OpenEmoji
