# [[Linux Firewall]]

---


[UFW (Uncomplicated Firewall)](https://wiki.archlinux.org/title/Uncomplicated_Firewall) if the most popular firewall manager, which depends on `iptables` or `nftables`.

Enable UFW need to firstly enable it in system service and run `ufw enable` (make sure `iptables.service` and other iptables-based service is **disabled**, see: [[Linux can't boot in after enter the password in display manager]]). Then, use UFW to filter network.
